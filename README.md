## 🧠 About Git
Git is a distributed version control system created by **Linus Torvalds** in 2005.  
It allows developers to track changes, maintain multiple versions of code, and collaborate efficiently across distributed environments.

Key features of Git include:

- Distributed architecture  
- Version tracking  
- Branching and merging  
- Local and remote repositories  
- High performance and reliability  

Git plays a central role in modern DevOps workflows and collaborative software development.
